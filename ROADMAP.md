# Atomic Habits Pro — Delivery Roadmap

Phased so that every phase ships something usable. Costs stay at $0 until Phase 4 (Claude API usage).

## Phase 0 — Foundation ✅ (done)

- [x] Next.js + TypeScript + Tailwind scaffold
- [x] Product vision, roadmap, data model docs
- [x] Design system: v1 look (Inter, indigo/violet, dark/light themes) as CSS tokens

## Phase 1 — Core app, local-first ✅ (done)

Rebuilt v1 features as React components with clean domain logic, storing data in localStorage first (no login needed to try the app):

- [x] Habit CRUD + frequency + skip/vacation (archive → Phase 2)
- [x] Today view: checklist, progress ring, XP/levels
- [x] Tracker calendar + heatmap + insights (strength, streaks, stats)
- [x] Goals, Journal, Scorecard, Stacks, Intentions, Contract
- [x] Rule-based Coach (ported from v1 as pure functions in `src/lib/logic.ts`)
- [x] v1 JSON import — same localStorage key and data shape, zero data loss

## Phase 2 — Accounts & sync (Supabase free tier)

- [ ] Supabase project + schema (see docs/DATA_MODEL.md) with Row Level Security
- [ ] Auth: Google, GitHub, email magic link
- [ ] Local ↔ cloud sync (offline-first, last-write-wins per day-cell)
- [ ] Cross-device continuity (phone + laptop)
- [ ] Deploy on Vercel

## Phase 3 — Notifications & PWA+

- [ ] Web push notifications via service worker + Supabase Edge Function cron
      (habit reminders at intention times, streak-at-risk warnings, weekly summary)
- [ ] Installable PWA with offline queue

## Phase 4 — Real AI Coach (Claude API)

- [ ] Weekly performance report generated by Claude from the user's stats
- [ ] Conversational coach chat grounded in the user's own data
- [ ] Habit design assistant ("I want to start running" → cue/stack/intention plan)
- [ ] Server-side only (Edge Function), rate-limited per user

## Phase 5 — Community & gamification+

- [ ] Accountability partners (habit contract co-signing, partner sees your streaks)
- [ ] Habit groups + community challenges
- [ ] Leaderboards (opt-in), seasonal events, coins/reward store

## Phase 6 — Expansion modules

- [ ] Health & wellness quick-logs (water, sleep, mood, steps)
- [ ] Reading tracker (books, sessions, pages)
- [ ] Digital wellness (no-phone hours, deep work sessions)
- [ ] PDF/Excel report exports
- [ ] Teams/organizations (roles, org dashboards)

## Out of scope until traction

Native iOS/Android (the PWA covers mobile until then) · enterprise SSO · wearable integrations (v2.0 list).
